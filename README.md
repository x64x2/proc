
    Prolog to C Compiler

    A minimal compiler translating Prolog to C code.

    ## Features
    - Compiled C output runs at native speed.
    - No bloat, just raw performance.  
    - Works on any system with a C compiler.

    ## Usage
    ```sh
    \$ prolog.c append.pl -o pltoc.c
    \$ gcc pltoc.c -o pltoc
    \$ ./pltoc
